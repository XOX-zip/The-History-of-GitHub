🤖 自动化神器

⚡ GitHub Actions

🔄 自动化 CI/CD 流水线 
name: 自动化工作流示例
on: [push, pull_request]
jobs:
  test:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v3
      - run: npm test
  deploy:
    needs: test
    runs-on: ubuntu-latest  
    steps:
      - run: npm run deploy

常用场景：

  • ✅ 自动测试运行
  • 🚀 自动部署发布
