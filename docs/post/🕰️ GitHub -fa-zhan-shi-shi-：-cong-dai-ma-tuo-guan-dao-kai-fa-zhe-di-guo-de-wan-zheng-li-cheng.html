## 功能识别挑战

```python
def mystery_feature():
    if code_changes:
        create_review()
        request_approval()
    return merge_if_approved()

这个代码描述的是哪个 GitHub 功能？

点击查看答案和解析

正确答案：Pull Requests

功能解析：
create_review() - 创建代码审查
request_approval() - 请求批准
merge_if_approved() - 批准后合并

这正是 Pull Request 的完整工作流程，是 GitHub 最核心的协作功能。

```
